# Support

Use the channel that best matches your question:

## Installation or onboarding

Open the installation/onboarding issue form if:

- `setup.py` does not complete
- `doctor.py --quick` fails on a fresh install
- hooks do not appear to load wiki context

## Bugs

Open the bug report form if something worked before and now behaves incorrectly.

## Ideas and feature requests

Open the feature request form for improvements, missing workflows, or UX suggestions.

## Questions and usage feedback

Use GitHub Discussions when enabled for:

- questions about architecture or intended workflows
- examples of how you are using the project
- broader product or documentation feedback

If Discussions are not enabled yet, open an issue with the closest matching form.
